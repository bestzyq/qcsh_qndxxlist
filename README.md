#### 介绍
作为团支书，你是否为需要登录青春上海平台查看同学学习情况而苦恼？  
看看这个项目吧！  
可以实时手动更新的排行榜，未学习同学提醒名单自动生成~  
解放你的双手！！！

#### 安装教程

1.  下载整个仓库，导入你的虚拟空间/服务器
2.  创建好数据库或者用qndxx_mysql_data.sql导入
3.  在所有access_token.php的account、password后面引号里面输入用户名、密码
```
$data = [
    "account" => "",
    "password" => "",
    "captchaId" => $id,
    "captcha" => $captcha
];
```
4.  在config.php输入数据库信息
5.  解放自我！
